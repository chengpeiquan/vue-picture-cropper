# 预设：固定尺寸

在某些业务场景下，可能需要将图片按照固定的尺寸裁剪，而不仅仅是按照固定的比例（例如：网站 Banner 推荐位），本插件提供了开箱即用的预设模式支持，只需要传入 `presetMode` 的相关选项。
