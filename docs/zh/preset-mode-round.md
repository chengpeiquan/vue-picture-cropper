# 预设：圆形

在某些业务场景下，可能需要将图片裁剪为圆形（例如：头像），本插件提供了开箱即用的预设模式支持，只需要传入 `presetMode` 的相关选项。

:::tip
请注意，圆形模式下，裁剪结果固定为 `.png` 文件。
:::
