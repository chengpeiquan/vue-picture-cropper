# CI Workflow Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Update GitHub Actions so pull requests run source-level verification and `main` pushes additionally run packed-package smoke tests before deployment.

**Architecture:** Refactor the existing workflow into separate `verify`, `smoke`, and `deploy` jobs. Use standard Node and pnpm setup actions, install Playwright Chromium in the jobs that need browser tests, and gate smoke plus deployment to `push` events on `main`.

**Tech Stack:** GitHub Actions, pnpm, Playwright, Node.js
