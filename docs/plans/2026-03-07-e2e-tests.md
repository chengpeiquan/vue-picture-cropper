# E2E Tests Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add Playwright-based e2e coverage for the core behaviors exposed by `src/`, using a dedicated local harness instead of the docs site.

**Architecture:** Add a standalone `e2e/` harness powered by Vite and Vue that imports the package source directly. Keep `vitest` for unit coverage and use Playwright only for real-browser verification of component and hook behaviors.

**Tech Stack:** Playwright, Vite, Vue 3, TypeScript
